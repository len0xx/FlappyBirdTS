# FlappyBirdTS

Игра Flappy Bird, написанная на TypeScript

### Компиляция и запуск

```bash
# Компиляция с помощью tsc
npm run tsc

# Установка простого веб-сервера NodeJS
npm install http-server

# Запуск сервера
http-server
```

После этого откройте браузер и перейдите по адресу `localhost:8080`
